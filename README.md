# Smart Expense Tracker

A full-stack MERN expense tracking application with analytics dashboard, authentication, budget management, and modern fintech UI.

## Features
- User Authentication
- Add/Edit/Delete Expenses
- Income Tracking
- Budget Management
- Dashboard Analytics
- Responsive UI
- Modern Fintech Design

## Tech Stack
- React.js
- Tailwind CSS
- Node.js
- Express.js
- MongoDB Atlas
- Render
- Vercel
